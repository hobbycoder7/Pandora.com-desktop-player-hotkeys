# Pandora.com desktop player hotkeys userscript  

## Keys
- J = Thumbs Down  
- K = Tired of Track  
- L = Thumbs Up  
- N = Next Track  

Only works on Now Playing page.  

For install information visit [Violentmonkey](https://violentmonkey.github.io/get-it/).
